# 股票短線分析助手

這是一個給新手使用的股票短線分析網頁，重點不是預測股價，而是幫助交易前整理：

- 趨勢
- 成交量
- 位置
- 風險
- 買點
- 停損
- 第一賣點

## 使用方式

打開 `index.html` 即可使用。

## GitHub Pages 建議設定

1. 進入 repository 的 Settings
2. 找到 Pages
3. Source 選擇 `Deploy from a branch`
4. Branch 選擇 `main`
5. Folder 選擇 `/root`
6. 儲存後等待 GitHub Pages 產生網址

## 注意

這個工具只做交易規劃輔助，不是投資建議。每一筆交易都應該先設定停損。
